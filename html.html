HTML

HyperText Markup Language

HTML (HyperText Markup Language) is the standard language used to create and design the structure of web pages on the internet. HTML uses a system of tags to define elements such as headings, paragraphs, links, images, and other content displayed by browsers.

HTML Tag Categories

Category Tag / Element Description
Document Structure !DOCTYPE, html, head, title, body Defines page structure and metadata
Text & Content h1-h6, p, br, hr, strong, em, a, img Headings, paragraphs, links, images, and formatting
Lists ul, ol, li Ordered and unordered lists
Layout div, span Containers for content, block and inline
Tables table, tr, th, td For tabular data structure
Forms form, input, textarea, button, label For collecting user input
