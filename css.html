CSS

Cascading Style Sheets

CSS is used to style HTML elements on web pages, controlling aspects such as colors, fonts, layout, spacing, and animations.

CSS Syntax & Parts

Part Description
Selector Specifies which HTML element(s) to style (p, .class, #id)
Property The aspect to modify (color, font-size, margin)
Value The setting for the property (red, 16px, etc.)
Declaration A property-value pair (color: red;)
Declaration Block One or more declarations wrapped in { }

Common CSS Property Categories

Category Description
Text Properties Control appearance, alignment, spacing, and decoration of text
Box Model Manage spacing, borders, padding, and margins of elements
Background Set color, image, position, and repetition of backgrounds
Layout & Display Control element positioning and visibility (flex, grid, block, inline)
Flexbox Create responsive layouts using flex container and item properties
Animation & Transition Add motion effects to elements using CSS properties
Effects & Visibility Apply transparency, shadows, filters, and visibility settings

CSS Grid System

The CSS Grid System allows you to create layouts with rows and columns, providing precise control for complex designs.

Part Description
Grid Container The parent element with display: grid, containing grid items
Grid Items Child elements inside the grid container
Grid Lines Dividing lines that define rows and columns
Grid Tracks Rows or columns formed between grid lines
Grid Cells Individual boxes inside the grid
Grid Area Space occupied by a grid item across rows and columns
Gap (Gutter) Spacing between grid items, controlled by gap, row-gap, column-gap
Nested Grids Grids placed within grids
Media Queries Adapt layouts for different screen sizes
